# BTO Upland Rovers Survey NN4103
## May 2026

This is a 3D map of OS grid square NN4103 for a British Ornithology Trust volunteer survey of upland birds. This is for the Upland Rovers project.

Made using minorua's <a href='https://github.com/minorua/Qgis2threejs'>Qgis2threejs</a> plugin for QGIS (3.34.4-Prizren).

The 3D map can be viewed <a href='https://calumleslie77.github.io/BTO_Survey/'>here</a>. The red arrows are the transects requested by the BTO.
